# Dotfiles

Using [dotbot](https://github.com/anishathalye/dotbot) with some
[Dotbot plugins](https://github.com/anishathalye/dotbot/wiki/Plugins)

## Installation

```bash
cd ~
git clone --recursive https://github.com/ycanty/dotfiles dotfiles
```
